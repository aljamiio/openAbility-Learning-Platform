# openAbility

openAbility is an HCI project and inclusive learning platform for deaf, deaf-mute, and visually impaired students in Bangladesh. Designed in Figma, it promotes accessible education through sign-language integration, audio learning tools, and user-friendly adaptive interfaces.

# HCI Prototype-figma

This is a code bundle for HCI Prototype-figma. The original project is available at
https://raven-echo-36456823.figma.site/
## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
